Story description     Hell Ride - A ride to remember...
You're a part-time reporter for The Tribune, the local newspaper. Earlier in the day, your editor called you and told you of a conversation he overheard between the Chief of Police and his Deputy about Whidbey Amusements. It seems there has been a rash of accidents and mishaps at the carnival. Sounds like there could be something suspicious going on. The editor wants you to collect evidence and write a hard hitting piece about this story. If you do a good job, maybe the paper will hire you on full-time.
As you explore the carnival, you learn (the hard way) that the Hell Ride attraction is malfunctioning with the potential for serious injuries to the riders. You must disable the ride off and prevent any loss of life... including your own.

Story file format:  Inform6/32/v3.1.2

The following use options are in force:

Set from the source text:

Use American dialect

Use serial comma

Use unabbreviated object names

Use dynamic memory allocation of at least 131072  Double this

Use maximum text length of at least 4096  Double this

Use scoring

Use maximum things understood at once of at least 500  Double this

Set from the Options.txt configuration file, or automatically :

Use index figure thumbnails of at least 50  Double this

Whereas these are not in force:

 ineffectual memory economy,  engineering notation,  predictable randomisation,  fast route-finding,  slow route-finding,  numbered rules,  telemetry recordings,  no deprecated features,  gn testing version,  command line echoing,  full-length room descriptions,  abbreviated room descriptions,  no scoring,  manual pronouns,  undo prevention,  VERBOSE room descriptions,  BRIEF room descriptions,  SUPERBRIEF room descriptions,  sequential action,  direct event handling,  persistent autosaves,

 See some technicalities for Inform maintainers only
